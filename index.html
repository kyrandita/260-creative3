This isn't intended to be a wow for you the user. It's a technically exciting thing, not a fun flashy thing to get everyones attention.

What I've designed here is a recursive directive that allows for infinite depth, allowing you to change types at any level at any time while also easily seeing the JSON representation of the data you've just created.

This is a problem I ran into at work recently and this serves as a proof of concept for what my coworker said couldn't be done. I've dressed it up a little for the purposes of this assignment, but as this is a proof of concept I don't need to once again prove I know how to write CSS.

If you play around with it for a little there are some glaring issues I didn't polish because I plan on doing that for the version for work which has an actual use case, this doesn't so I didn't waste the time.
